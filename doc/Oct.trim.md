## trim or Oct.trim()

### Description

Delete the space in the first and end of a string.

### Syntax
	Oct.trim(str)  => value

	str.trim()  => value

### Demo

1.Deal with "   sss   "

	console.log("sss".length);
	console.log(Oct.trim("   sss   "));
	console.log(Oct.trim("   sss   ").length);
	console.log("   sss   ".trim().length);
